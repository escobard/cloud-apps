# API

## Quickstart

Starts the application on `http://localhost:4000`

1. `npm install`
2. `npm start`

## Usage

### Development

`npm run dev`

### Develop unit tests

`npm run test-watch`

### Unit tests

`npm run test`

### Production

`npm start`

### With Docker
`npm run docker`
