# UI

For more UI documentation, visit [documents/ui.md](documents/ui.md)

## Usage

### Installation

`npm install`

### Local development

`npm start`

### Local testing

To run all tests, with coverage:

`npm test`

To run a single test:

`npm run test-single /path/to/tests`

### Lint

To check for lint errors:

`npm run lint`

To fix lint errors

`npm run lint-fix`

### Production

`npm build`
