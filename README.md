# cloud-apps

A sample cloud-native product, designed to scale.
  
## Quickstart

[Node.js](https://nodejs.org/en/), [npm](https://www.npmjs.com/), [Docker](https://www.docker.com/) and [Docker compose](https://docs.docker.com/compose/) must be installed.
        
#### Development        
 `npm run dev`            

#### End to end tests  
  `npm run test`  
  
#### Production        
 `npm run start`
