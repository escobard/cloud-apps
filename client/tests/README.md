# End to End Tests

## Quickstart

This application expects the ui to be running on `localhost:3000` and the api to be running on `localhost:4000`, connected to postgres on `localhost:5432`.

1. `npm install`
2. `npm start`

## Usage

#### Run tests

`npm run test`

### Watch tests

`npm run test-watch`

#### Tests in headless mode

`npm run test -- --headless`

### With Docker

`npm run docker`
