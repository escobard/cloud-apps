# UI

## Quickstart

Starts the application on `http://localhost:3000`.

1. `npm install`
2. `npm start`

## Usage

### Development

`npm start`

### Develop unit tests

`npm run test-watch`

### Unit tests

`npm test`

### Lint

`npm run lint`

#### Lint fix

`npm run lint-fix`

### Production

`npm run build`

### With Docker
`npm run docker`
